0. Materi Dasar Vektor

Definisi

Vektor adalah besaran yang memiliki nilai dan arah. Di tingkat SMA/XI, kita biasanya mempelajari vektor di bidang (R2) dan ruang (R3).

Vektor posisi dinyatakan dengan: v = xi + yj (R2) atau v = xi + yj + zk (R3), di mana i, j, k adalah vektor satuan.

1. Penjumlahan Vektor

v₁ + v₂ = (x₁ + x₂)i + (y₁ + y₂)j

Contoh 1.1

v₁ = 2i + 3j, v₂ = 4i − j

help_outline Lihat Cara
Jumlahkan komponen yang sejenis (i dengan i, j dengan j):
= (2 + 4)i + (3 + (-1))j
= 6i + 2j
Hasil: 6i + 2j
Contoh 1.2

v₁ = -3i + 5j, v₂ = 2i + j

help_outline Lihat Cara
= (-3 + 2)i + (5 + 1)j
= -1i + 6j
Hasil: -i + 6j

2. Pengurangan Vektor

v₁ − v₂ = (x₁ − x₂)i + (y₁ − y₂)j

Contoh 2.1

v₁ = 5i + 2j, v₂ = 3i + 4j

help_outline Lihat Cara
= (5 - 3)i + (2 - 4)j
= 2i - 2j
Hasil: 2i - 2j
Contoh 2.2

v₁ = 4i − j, v₂ = −2i + 3j

help_outline Lihat Cara
Perhatikan tanda negatif:
= (4 - (-2))i + (-1 - 3)j
= (4 + 2)i - 4j = 6i - 4j
Hasil: 6i - 4j

3. Perkalian Skalar (k · v)

k · v = (k · x)i + (k · y)j

Contoh 3.1

v = 2i - 3j, k = 2

help_outline Lihat Cara
Kalikan angka 2 ke setiap komponen:
2(2i) - 2(3j) = 4i - 6j
Hasil: 4i - 6j
Contoh 3.2

v = -i + 4j, k = -3

help_outline Lihat Cara
-3(-1i) + (-3)(4j) = 3i - 12j
Hasil: 3i - 12j

4. Dot Product (Perkalian Titik)

v₁ · v₂ = x₁x₂ + y₁y₂

Contoh 4.1

v₁ = 2i + j, v₂ = i − 3j

help_outline Lihat Cara
Hasil perkalian titik adalah angka tunggal (skalar):
= (2 * 1) + (1 * -3) = 2 - 3 = -1
Hasil: -1
Contoh 4.2

v₁ = 3i − 2j, v₂ = 4i + j

help_outline Lihat Cara
= (3 * 4) + (-2 * 1) = 12 - 2 = 10
Hasil: 10

5. Besar Vektor (Panjang)

|v| = √(x² + y²)

Contoh 5.1

v = 6i − 8j

help_outline Lihat Cara
|v| = √(6² + (-8)²) = √(36 + 64) = √100 = 10
Hasil: 10
Contoh 5.2

v = 3i + 4j

help_outline Lihat Cara
|v| = √(3² + 4²) = √(9 + 16) = √25 = 5
Hasil: 5

6. Sudut Antar Dua Vektor

cos θ = (v₁ · v₂) / (|v₁| |v₂|)

[Image of angle between two vectors]
Contoh 6.1

v₁ = (1,1), v₂ = (0,1). Berapa sudut θ?

help_outline Lihat Cara
1. v₁ · v₂ = (1*0) + (1*1) = 1
2. |v₁| = √(1²+1²) = √2, |v₂| = √(0²+1²) = 1
3. cos θ = 1 / (√2 * 1) = 1/√2 = ½√2
4. θ = 45°
Hasil: 45°

7. Perkalian Silang (Cross Product)

a × b = Vektor Baru (Tegak Lurus)

[Image of vector cross product right hand rule]
Contoh 7.1

a = i, b = j. Berapa a × b?

help_outline Lihat Cara
Sesuai aturan siklus vektor satuan:
i × j = k
j × k = i
k × i = j
Hasil: k
